Ubuntu Server

The Ubuntu Server is virtualized within the main Proxmox host and hosts Docker containers for ease of deployment.
Virtualization of the Ubuntu host increases segmentation of services, decreasing possible attack vectors, and allows snapshotting of the server.

Docker Containers

    Immich - Open source Google Photos alternative.
    Jellyfin - Open source media library
    Minecraft Server - Video Game server accessed through FQDN via Minecraft Client
